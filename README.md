# 基于 ESOL 数据集的分子溶解度预测

本项目是一个入门级的 AI 药物发现任务，用深度学习模型预测小分子在水中的溶解度（log mol/L）。

溶解度是药物研发中重要的物理化学性质：
- 溶解度低的药物可能口服吸收差，难以达到有效血药浓度；
- 溶解度高的药物更容易制成制剂，但也可能带来渗透性问题。

本项目基于 ESOL 数据集（含水溶解度），通过分子的 SMILES 字符串以及多种分子描述符，训练一个深度神经网络（DNN）来预测分子的 log 溶解度。

---

### 1. 数据处理
- **输入数据**：`esol.csv`，包含以下关键列：
  - 特征：
    - `smiles`：分子的 SMILES 字符串表示。
    - 额外分子描述符：`Minimum Degree`、`Molecular Weight`、`Number of H-Bond Donors`、`Number of Rings`、`Number of Rotatable Bonds`、`Polar Surface Area`。
  - 标签：`measured log solubility in mols per litre`：实验测得的 log 溶解度（目标变量）。
- **数据划分**：训练集:验证集:测试集 = 8:1:1，随机划分，保证数据分布一致。

### 2. 分子特征工程
- **Morgan 指纹（半径=2，2048 维）**：将分子结构转换为固定长度的二进制向量，捕捉原子周围局部子结构信息。
- **额外分子描述符**：与指纹向量拼接，提供分子整体性质（分子量、氢键供体数、环数、可旋转键数、极性表面积、最小原子度）。
- **标准化**：对额外描述符进行 Z-score 标准化（基于训练集拟合）。

### 3. 模型结构（DNN）
2 层全连接网络（1 个隐藏层 + 1 个输出层）：
- 隐藏层：`Linear(2054, 16)`  输入维度 = 2048（指纹）+ 6（描述符）= 2054
- ReLU 激活
- Dropout 层：`Dropout(0.6)` 
- 输出层：`Linear(16, 1)`

### 4. 训练与评估
- **分批训练（Mini-batch）**：使用批量大小为 64 的 mini-batch 梯度下降。通过 `DataLoader` 将训练集划分为多个批次，在每个批次上计算损失并更新参数，既能加速收敛，又能提升训练稳定性。
- **损失函数**：均方误差（MSE）
- **优化器**：Adam（学习率 0.001，权重衰减 1e-2）
- **评估指标**：
  - **RMSE**（均方根误差）：与原始数据单位一致（log mol/L），越小越好
  - **R²**（决定系数）：模型解释的方差比例，越接近 1 越好 （验证集 R² 达到最好时，模型的权重参数会用于测试集求出R²和RMSE）
- **正则化**：
  - Dropout 层（0.6）
  - L2 正则化（weight_decay=1e-2）
  - 早停（Early Stopping）：监控验证集 R²，`patience=10` 轮无改善则停止
- **训练过程可视化**：绘制训练集/验证集的 RMSE 和 R² 曲线，观察收敛与过拟合情况。

### 5. 最终性能（示例）
- **测试集 RMSE**：约 1.03
- **测试集 R²**：约 0.80
- **单个分子预测示例**：
  - 分子：2,3,3',4,4'6-PCB（多氯联苯）
  - 真实溶解度：-7.66（log mol/L）
  - 预测溶解度：约 -7.4842（接近真实值）
